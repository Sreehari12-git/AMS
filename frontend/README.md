# AMS – Attendance Monitoring & Leave Management System

Attendify is a full stack Attendance Monitoring System built using React, Express.js, PostgreSQL, and Prisma ORM.

The application helps organizations manage:
- Employee attendance
- Leave requests
- Role-based access
- Attendance history
- Geofencing-based attendance

---

# Tech Stack

## Frontend
- React
- React Router DOM
- Redux Toolkit
- Axios

## Backend
- Node.js
- Express.js

## Database
- PostgreSQL (Supabase / Docker)
